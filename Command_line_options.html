Command line options

From CompuCell3D Wiki

Jump to: navigation, search

It is possible to specify various parameters when starting the cc3D script(s). To see what those options are, one can run: 

$ compucell3d.sh -h    (or, --help)
...
--------------------------------------------------------
USAGE: ./compucell3d.sh -i <sim file (.cc3d or .xml or .py)> -s <ScreenshotDescriptionFile> -o <custom outputDirectory>
-w <widthxheight of graphics window>
--exitWhenDone   close the player after simulation is done
--noOutput   ensure that no screenshots are stored regardless of Player settings
--prefs    name of preferences file to use/save
-p    playerSettingsFileName (e.g. 3D camera settings)
-h or --help   print (this) help message

e.g.  compucell3d.sh -i Demos/cellsort_2D/cellsort_2D/cellsort_2D.cc3d -w 500x500 --prefs myCellSortPrefs

Through this investigation, I learned that we were checking for command line options in two places: player/compucell3d.pyw and player/Plugins/ViewManagerPlugins/SimpleTabView.py (def processCommandLineOptions).

I've now edited this so that we only check for command line options one place, in compucell3d.pyw (and then pass the options to the processCommandLineOptions function in SimpleTabView.py.


AMD_script - refer to this page that discusses how we dump 3D camera parameters to a file and then indicate that that file should be used to set the camera on the command line.
