New QSettings

startup_params - for older wiki page on saving Prefs

We now use the QSettings.IniFormat format instead of the .NativeFormat format. This means the cc3D Preferences file will be saved locally in a user's directory, instead of some system-level directory/registry. And this will offer us a better way to provide cc3D simulation-dependent preferences.

For more information, rf http://www.riverbankcomputing.co.uk/static/Docs/PyQt4/html/qsettings.html#Format-enum

For now, we simply make it possible for the user to specify a new/existing Prefs filename on the compucell3d startup script. If none is specified, then the default file "cc3d_default.ini" will be used. However, it is now possible to specify a different file via:

compucell3d.sh --prefs=my_cool_sim_prefs

The preferences filename will be shown in a the Output tab of the CC3D Preferences dialog; however, the widgets to change it during a simulation are currently disable. Eventually, we make them functional so that a user could change the preferences from within the GUI.
