Python debugging

Try the following from your CC3D build directory (Linux/OSX): 

% cp runScript.sh setup.sh

Edit setup.sh and comment out the 'python' line (next to last) with a '#' preceding it:
#python ${PREFIX_CC3D}/player/CompuCellPythonSimulationCML.py $* --currentDir=${current_directory}

% source setup.sh    # this will simply set the proper env vars for a CC3D run


Run the CC3D Python script with the Python debugger (pdb):
